About This Project

This project provides a complete machine learning solution to classify movie reviews as positive or negative. It uses Natural Language Processing (NLP) techniques to understand and interpret the sentiment expressed in text.

Built with an MLOps-first approach, the application showcases a full model lifecycle: from data preprocessing and hyperparameter tuning with Optuna, to experiment tracking with MLflow and deployment via a Flask API. The goal is to deliver not just accurate predictions, but a robust and reproducible system.
